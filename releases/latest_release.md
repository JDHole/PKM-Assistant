# PKM Assistant v1.0.9

## Co nowego

### Singleton usuniety - koegzystencja z Smart Connections
PKM Assistant nie uzywa juz globalnego `window.smart_env`. Mozesz miec zainstalowane oba pluginy jednoczesnie bez konfliktow.

### Embeddingi wlaczone
Indeksowanie vaulta dziala domyslnie. Notatki sa automatycznie embedowane przy starcie pluginu.

### Semantyczny vault_search
Agent wyszukuje notatki semantycznie (po znaczeniu), nie tylko po dokladnym dopasowaniu tekstu. Szukasz "wakacje" - znajdzie notatke o "urlop nad morzem".

### Semantyczny memory_search
Wyszukiwanie w pamieci agenta rowniez korzysta z embeddingowego wyszukiwania gdy dostepne.

### Rebranding
Usuniete pozostalosci Smart Connections z UI - linki, nazwy widokow, "What's New" popup.
